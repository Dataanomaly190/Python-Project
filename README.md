# Python-Project

A comprehensive collection of Python projects ranging from Data Analytics to Computer Vision and AI Agents.

## Repository Structure

### [Artificial Intelligence & Agents](./Jerry/)
- **Jerry**: A sophisticated AI agent with local LLM integration (Qwen, Phi-4) and multi-brain switching capabilities.

### [Face Detection & Recognition](./FaceDetectionSystem/)
- **Trilogy Suite**: 
  - **Basic**: Haar Cascade detections in static images.
  - **Trainable**: LBPH model with custom dataset collection and training.
  - **Pretrained**: DeepFace-powered high-accuracy recognition using CNN embeddings.

### [Data Analytics & Dashboards](./DataAnalytics/)
- **Bank Marketing**: Z-score statistical analysis and demographic conversion tracking.
- **Books Analysis**: Temporal popularity scoring and multi-dimensional visualization.
- **Census Report**: Socioeconomic probability modeling and labor-age banding.

### [Automations & Scripts](./Youtube_Playlist-Transfer/)
- **Youtube Playlist Transfer**: CLI tools for migrating media between accounts.

---

## Getting Started
Each project directory contains its own `requirements.txt` and a detailed `README.md` explaining setup and technical execution.

## Main Tech Stack
- **AI/ML**: TensorFlow, Keras, DeepFace, NumPy
- **CV**: OpenCV (LBPH, Haar Cascades)
- **Data**: Pandas, Matplotlib, Seaborn
- **UI**: Streamlit (Reactive Web Apps)
